# Приют для животных "Groomify"
Учебный проект по прикладному программированию.
Приложение на **WPF** для управления карточками животных и заявками на адопцию

## Использованные технлогии
* **Язык:** C#
* **Интерфейс:** XAML (WPF)
* **База данных:** SQL Server 2022
* **Архитектура:** Модели данных (Models) и статические сессии

## 📌 Как запустить
1. Склонируйте репозиторий.
2. Откройте файл `.sln` в Visual Studio 2022.
3. Проверьте строку подключения в `App.config`.
